# Compensation Policy

## Overview

We are committed to customer satisfaction. When service failures occur, we offer compensation based on the severity and impact of the issue. Our support agents are empowered to resolve issues quickly using the following guidelines.

## Late Delivery Compensation

### Standard Shipping Delays
- Delivered 1-3 days late: 10% discount code for next order
- Delivered 4-7 days late: 15% discount code for next order + free standard shipping on next order
- Delivered 8+ days late: 25% discount code for next order + free express shipping on next order
- Not delivered within 14 days of estimated date: Full refund or free replacement (customer's choice)

### Express Shipping Delays
- Delivered 1 day late: Refund of shipping cost difference (express minus standard)
- Delivered 2+ days late: Full refund of shipping cost + 10% discount code
- Not delivered within 7 days: Full refund or free replacement with overnight shipping

### Overnight Shipping Delays
- Delivered 1 day late: Full refund of shipping cost + 15% discount code
- Delivered 2+ days late: Full refund of shipping cost + 25% discount code
- Not delivered within 3 days: Full refund or free replacement (customer's choice)

## Damaged or Defective Items

- Item arrived damaged: Full refund or free replacement + free return shipping
- Item defective within warranty: Free replacement or repair + free shipping both ways
- Item defective outside warranty: 30% discount on replacement purchase

## Wrong Item Received

- Wrong item sent: Free return shipping + correct item with express shipping at no charge
- Wrong size/color (customer error): Standard return policy applies
- Wrong size/color (our error): Free return shipping + correct item with express shipping + 10% discount code

## Service Failure Compensation

- Website error causing incorrect charge: Immediate refund of overcharge + 10% discount code
- Customer service response time exceeding 24 hours: 10% discount code
- Multiple failed delivery attempts due to our error: 20% discount code + free shipping on next 3 orders

## Loyalty Considerations

For customers with 5+ previous orders:
- Compensation tiers are upgraded by one level
- Additional 5% added to any discount code
- Priority processing for replacements

For customers with 10+ previous orders:
- Double the standard compensation
- Dedicated support representative
- Free overnight shipping on replacements

## Discount Code Terms

- Discount codes are valid for 90 days from issue date
- One discount code per order unless otherwise specified
- Discount codes cannot be combined with sale prices
- Discount codes are non-transferable and single-use
- Maximum discount capped at $100 per code

## Escalation

If a customer is unsatisfied with the offered compensation:
1. Agent may increase compensation by one tier without manager approval
2. For further escalation, transfer to a senior support agent
3. Senior agents may offer up to 50% refund at their discretion
4. Full refund exceptions require manager approval
